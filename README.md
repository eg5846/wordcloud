# wordcloud
Wordcloud REST service

## Requirements
* git client
* openjdk version "16"
* Optional: Apache Maven 3.8.1
* Optional: IntelliJ IDEA Ultimate (ideaIU-2021.1.1.tar.gz)

## Build app
```
$ git clone git@github.com:eg5846/wordcloud.git

$ cd wordcloud

$ ./mvnw clean test install
```
